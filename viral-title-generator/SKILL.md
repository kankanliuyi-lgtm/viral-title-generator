# X爆款标题生成器

基于116条真实X/Twitter爆款帖子（阅读量10万-260万+）提炼的12种标题公式，帮助AI自媒体创作者生成高点击率标题。

## 使用流程

收到用户的标题生成请求后，按以下步骤执行：

### Step 1：理解内容

从用户输入中提取以下信息：
- **内容主题**：用户要写什么（AI工具评测？赚钱方法？教程？行业观点？）
- **核心卖点**：这条内容最大的亮点是什么
- **目标平台**：X/公众号/视频号/抖音（不同平台风格微调）
- **内容类型**：工具推荐、教程、案例拆解、观点输出、资源分享、变现方法

如果用户没有明确说明以上信息，根据已有内容合理推断，不要反复追问。

### Step 2：匹配标题套路

根据内容类型，从12种套路中推荐2-3种最匹配的。匹配逻辑参考 `references/title_formulas.md`。

匹配优先级规则：
- 工具评测/推荐 → 优先：工具发现型、震惊情绪型、数字清单型
- 教程/SOP → 优先：教程保姆级型、数字清单型、故事案例型
- 赚钱/变现 → 优先：利益诱惑型、故事案例型、公式等式型
- 行业观点/趋势 → 优先：趋势认知型、争议吐槽型、对比VS型
- 资源/福利分享 → 优先：紧急行动型、数字清单型、工具发现型
- 案例拆解 → 优先：故事案例型、震惊情绪型、利益诱惑型
- 产品吐槽/评测 → 优先：争议吐槽型、对比VS型、禁忌边界型

### Step 3：生成候选标题

每种推荐套路生成3个候选标题，共6-9个候选。

生成时遵循以下原则：

**必须做到：**
- 标题控制在15-50字（X平台最佳长度）
- 至少包含一个具体数字（金额、数量、倍数、天数）
- 使用口语化表达，像朋友聊天而不是新闻报道
- 包含至少一个可搜索的关键词（工具名、产品名、平台名）
- 制造信息差感："你不知道的""隐藏的""别人不会告诉你的"

**高分技巧——套路混搭：**
真实爆款标题往往是2种套路的混合体。混搭组合参考：
- 数字清单 + 禁忌边界 = "这10个ChatGPT提示词感觉非法（但不违法）"
- 震惊情绪 + 工具发现 = "卧槽！这个工具绝了！OfficeCLI给AI装外挂"
- 利益诱惑 + 公式等式 = "自媒体公众号 + AI邪修 + 私域 = 月入10w+"
- 故事案例 + 数字清单 = "我朋友用ChatGPT减掉了20公斤...7条指令如下"
- 教程保姆级 + 震惊情绪 = "爆肝制作，全网最细Claude Code零基础系统教程"
在9个候选标题中，至少安排2-3个混搭标题。

**绝对避免：**
- 学术化/官方化表达："浅析""探讨""论述"
- 没有具体信息的空洞标题："AI太强了""这个工具很好用"
- 过长标题（超过60字在X上会被截断）
- 纯英文标题（目标受众是中文用户）
- 标题党但无法兑现的承诺

**平台风格微调：**
- **X/Twitter**：可以更口语、更情绪化，感叹号和省略号多用，"卧槽""笑死"可以出现
- **公众号**：稍正式一点，但不能无聊，重点放在"信息差"和"具体利益"
- **视频号/抖音**：更短更直接，前5个字决定生死，动作词开头（"教你""拆解""揭秘"）

### Step 4：标题评分

对每个生成的标题进行5个维度评分（每项0-20分，满分100）：

| 维度 | 评分标准 | 权重 |
|------|---------|------|
| 好奇心 | 是否让人想点进来看？有没有信息差/悬念/意外？ | 20分 |
| 具体性 | 有没有具体数字、工具名、金额？能不能一眼看出内容是什么？ | 20分 |
| 情绪感 | 有没有引发情绪反应？惊讶/兴奋/愤怒/共鸣？ | 20分 |
| 可搜索性 | 包含多少可搜索关键词？SEO友好度？ | 20分 |
| 行动驱动 | 看完标题是否有立刻行动的冲动？收藏/转发/点击？ | 20分 |

评分规则：
- 90-100分：极品标题，直接用
- 75-89分：优秀标题，微调后可用
- 60-74分：及格标题，需要优化
- 60分以下：不及格，重写

## 标题优化模式

如果用户给的不是"生成新标题"，而是"优化已有标题"，切换到优化模式：

1. 先对用户现有标题评分（5维度）
2. 诊断问题：哪个维度最弱？
3. 推荐1-2个最适合的套路进行改写
4. 给出3个优化版本，对比评分变化
5. 说明"改了什么""为什么这样改"

## 标题自检清单

生成标题后，用以下清单自检，确保每个标题至少满足5条以上：

- [ ] 有具体数字吗？（金额、数量、倍数、天数）
- [ ] 前5个字能抓住注意力吗？
- [ ] 包含可搜索的关键词吗？（工具名/产品名/平台名）
- [ ] 读起来像人说话吗？（不像新闻稿/论文）
- [ ] 有信息差/悬念感吗？（让人想知道更多）
- [ ] 有情绪触发词吗？（卧槽/笑死/绝了/太野了）
- [ ] 承诺了具体利益吗？（学到什么/赚多少/省多少时间）
- [ ] 长度合适吗？（15-50字最佳）
- [ ] 能一眼看出内容类型吗？（教程/评测/案例/观点）
- [ ] 发出去你自己会想点进去看吗？

达标标准：满足7条以上 = 可发布，满足5-6条 = 需优化，5条以下 = 重写。
