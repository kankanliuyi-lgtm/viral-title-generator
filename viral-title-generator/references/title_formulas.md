## 1. 数字清单型

**核心公式：** `[数字] + [工具/方法] + [具体利益]`

**变体公式：**
- `[N]个[工具名][功能词]，[利益承诺]`
- `如果你有[低门槛条件]，这是[N]个[方法]，[收益]`
- `[时间/场景词]...[N]个[资源]建议[行动]`

**真实案例（按阅读量排序）：**
- 我朋友用ChatGPT减掉了20公斤...7条指令如下（162万）
- 如果你有笔记本+WiFi+时间，这是10个Claude提示词，每天赚500美元（62万）
- 这10句决定性强力句子，让Claude/ChatGPT/Gemini智能提升20倍（12万）
- 这10个ChatGPT提示词感觉非法（但不违法）（42万）
- 每个人必装的5个Codex神级Skill（5.6万）
- 免费开源10个真正能变现的AI提示词（证件照等）
- 7个Gemini提示词让你学得快得多
- 每天信息太杂...这4个免费网站建议收藏（15万）

**关键技巧：**
- 数字优先用 3/5/7/10，奇数更有吸引力
- 利益要量化："提升20倍" > "大幅提升"
- 叠加门槛限定词降低心理压力：免费、必装、每个人
- 可混搭"禁忌感"：感觉非法（但不违法）
- "...如下"制造即刻获取感

**适用内容类型：** 提示词分享、工具推荐、资源合集、技巧总结

## 2. 震惊情绪型

**核心公式：** `[情绪词] + [悬念/发现] + [具体内容]`

**变体公式：**
- `[情绪词]，[发现的事物]！[工具/产品名]`
- `[情绪词]！[身份群体]的[夸张比喻]来了！[产品名]`
- `[情绪词]，[具体行为描述]`

**情绪词库（按冲击力排序）：**
- 第一梯队（最强）：卧槽、笑死、Damn、牛逼
- 第二梯队（强）：绝了、太疯狂了、太高级了、太野了
- 第三梯队（中）：真的牛、厉害了、惊呆了

**真实案例：**
- 笑死，这个图谁整的呀（258万）
- 卧槽，自媒体人的核武器来了！AiToEarn开源神器（8.5万）
- 卧槽，我的Agent连上真实手机跑小红书
- 卧槽！这个工具绝了！OfficeCLI给AI装外挂（10.8万）
- 卧槽，这个玩的也太高级了！3D城市模型生成（11.3万）
- Damn！所有AI开发者，立刻停下npm install（57万）
- 牛逼！自媒体人的开挂神器来了！AiToEarn

**关键技巧：**
- 情绪词必须放在最开头，模拟脱口而出的真实反应
- 感叹号不能省
- 情绪词后面紧跟"发现"或"意外"
- 具体内容要跟上，不能光有情绪没有信息

**适用内容类型：** 新工具发现、炸裂效果展示、意外发现分享

## 3. 利益诱惑型

**核心公式：** `[低门槛条件] + [具体收益数字] + [方法/工具]`

**变体公式：**
- `如果你有[条件]，[方法]，[日/月收益]`
- `从此打开[赚钱模式]！[工具名]`
- `[年份]最被低估的[副业类型]：[方法]`
- `[虚拟/实物产品]...[平台]卖赚[金额]`

**真实案例：**
- 如果你有笔记本+WiFi+时间...每天赚500美元（62万）
- 抖音小游戏，流量主收益每天1000+（83万）
- 2026年最被低估的副业：短视频套利（18万）
- 虚拟产品之——PS全套插件包...闲鱼小红书卖赚几十万（9.2万）
- 自媒体公众号+AI邪修+私域=月入10w+（4.8万）
- 从此打开躺赚模式！AiToEarn AI内容赚钱智能体（7.9万）
- 真的可以用信息差赚钱！美国国防部UFO文件...（2.5万）

**金额表达参考：**
- 日收入：每天500美元、日入1000+
- 月收入：月入10w+、月销20万美金
- 总收入：赚几十万、3000元成本爆火

**关键技巧：**
- 金额必须具体，模糊的"赚很多"无效
- 门槛越低越好：笔记本+WiFi、免费工具、零成本
- 关键词组合：躺赚、副业、套利、信息差、变现
- 暗示"别人已经在赚了"的紧迫感

**适用内容类型：** 副业推荐、变现方法、赚钱案例、信息差分享

## 4. 争议吐槽型

**核心公式：** `[争议观点/吐槽] + [具体对象] + [情绪放大]`

**变体公式：**
- `[产品名]现在是[贬义类比] [吐槽词]`
- `[事物]完犊子了：[原因]`
- `[产品名]真的好[口语评价]`
- `[时间对比]...[吐槽现象]`

**真实案例：**
- 豆包淫叫，你们一定要听完（267万）
- gemini现在是美国版豆包 中看不中用（45万）
- 22年刷推特每日大赛惊喜...现在全是AI人机（85万）
- 知识产权完犊子了：AI生成专利/软著Skill开源
- grok真的好骚聊天跟看黄文一样
- 海南三亚皮皮虾事件...网暴吃人不吐骨头（5.7万）

**口语吐槽词库：**
- 产品差评：中看不中用、完犊子了、不行
- 调侃：好骚、太野了、思路清奇
- 类比拉踩：美国版豆包、翻版XX

**关键技巧：**
- 吐槽对象要有知名度，吐槽小众产品没流量
- 口语化 > 正式措辞："完犊子了" > "存在重大缺陷"
- 制造"说出了大家想说但不敢说的话"的共鸣
- 注意尺度，过分人身攻击会翻车

**适用内容类型：** 产品评测、行业吐槽、热点评论

## 5. 故事案例型

**核心公式：** `[人物] + [反差/意外结果] + [方法暗示]`

**变体公式：**
- `我朋友用[工具][做到了XX]...[N]条[方法]如下`
- `这哥们[反差行为]！[身份反差][意外结果]`
- `[作品名]：[普通人身份][低成本][爆火结果]`
- `[大公司/名人]为了[目的]有多[状态]...`

**真实案例：**
- 我朋友用ChatGPT减掉了20公斤...7条指令如下（162万）
- 这哥们思路太野了！00后哈工大毕业辞职做亚马逊（161万）
- AI短剧《丧尸清道夫》：中专生10天3000元成本爆火
- 腾讯为了入股DeepSeek有多卑微...想要认购20%直接被拒（89万）
- 干货分享：到底如何做爆款？4月全网涨粉6万+（18万）

**人物身份反差组合：**
- 草根逆袭：中专生、00后、我朋友（强调普通人）
- 大佬翻车/卑微：腾讯被拒、Sam Altman转发
- 自己的故事：干货分享、我的Agent

**关键技巧：**
- 人物身份越普通/意外，反差越大
- 结果要数字化：减20公斤、月销20万、3000元成本
- "我/我朋友"增强真实感
- 省略号"..."制造悬念，暗示还有更多信息

**适用内容类型：** 案例拆解、人物故事、个人经验分享

## 6. 教程保姆级型

**核心公式：** `[降低门槛词] + [具体技能] + [效果承诺]`

**变体公式：**
- `[努力词]，全网最细[工具名][零基础]系统教程`
- `[技能]保姆级小白入门教程`
- `用[工具]做[成果]SOP：[效果数据]`
- `[技能]看[N]个[资源]就够`

**真实案例：**
- 爆肝制作，全网最细Claude Code零基础系统教程（33万）
- Codex保姆级小白入门教程
- 用AI做短剧解说SOP：300万点赞爆款拆解
- 自学 Vibe Coding 看这三个开源项目就够（10万）
- 使用故事版做1分钟AI微电影：Gemini+gpt-image2+seedanc（38万）

**降低门槛词库：**
- 难度降低：保姆级、小白、零基础、入门
- 完整性：全网最细、系统教程、全套打法、完整SOP
- 极简性：看这个就够、一句话生成、3步搞定
- 努力背书：爆肝制作、花了一个月整理

**关键技巧：**
- "保姆级""零基础"直接打消"我学不会"的顾虑
- "全网最细"暗示比别人的教程更有价值
- 工具链罗列增强专业感（但不要超过3个）
- 配合效果数据：300万点赞、涨粉6万

**适用内容类型：** 使用教程、操作指南、学习路径

## 7. 工具发现型

**核心公式：** `[发现感] + [工具名] + [核心能力一句话]`

**变体公式：**
- `在GitHub挖到一个好用的工具：[工具名][一句话能力]`
- `一个超酷AI工具：[工具名][核心功能]`
- `有人为[平台]开发了[免费][功能]工具...太疯狂了`
- `[工具名]！[属性]AI[功能]工具`

**真实案例：**
- 在GitHub挖到一个好用的工具：xiaohongshu-mcp自动化小红书（2.3万）
- 一个超酷AI工具：Video Podcast Maker一句话生成专业视频
- 有人为Claude Code开发了免费视频编辑工具...太疯狂了（7.5万）
- App Icon Maker！开源AI macOS图标生成工具（2.5万）
- 打破AI信息差：TrendRadar自动监控全网热点（20万）

**发现感词库：**
- 挖到、发现、刷到、找到了
- "有人开发了..."（第三方视角增强可信度）
- "终于有了...""竟然有这个..."

**关键技巧：**
- 工具名必须出现，增强可搜索性和专业感
- 一句话说清核心能力："一句话生成""自动监控"
- "免费""开源"是强加分词
- "GitHub"对技术向受众是信任加分

**适用内容类型：** 工具推荐、GitHub项目安利、产品介绍

## 8. 趋势认知型

**核心公式：** `[类比/反常识] + [趋势判断] + [行动指引]`

**变体公式：**
- `AI就像[历史类比]：现在是[判断]`
- `[数据反差]：[趋势结论]`
- `其实[反常识观点]：[支撑论据]`
- `[时代词]，[群体]最该[行动]的不是[常见选择]`

**真实案例：**
- AI就像19年短视频：现在是最大红利期
- 有人总是担心这波存储，踏空了...AI时代才刚刚开始（100万+）
- 硬件涨10倍，软件还没动：AI超级周期下半场机会（22万）
- AI时代，自媒体人最该盯的不是更多写作工具（3.8万）
- 其实大环境越差，越要反其道（35万）

**类比模板：**
- AI就像[XX年的短视频/移动互联网/电商]
- 这就像[历史事件]重演
- [行业]正在经历[历史类比]时刻

**关键技巧：**
- 历史类比降低理解门槛：像19年短视频
- 反常识观点制造认知冲突
- 暗示"再不行动就晚了"
- 适合有一定粉丝基础的账号

**适用内容类型：** 行业分析、趋势预判、认知分享

## 9. 公式等式型

**核心公式：** `[元素A] + [元素B] + [元素C] = [结果]`

**变体公式：**
- `[方法A] + [方法B] + [方法C] = [收益]`
- `[属性] + [属性] + [属性] = [概念]`
- `[工具] + [N]个[资源]`

**真实案例：**
- 美貌 + 性感 + 免费 = 流量密码（18万）
- 自媒体公众号 + AI邪修 + 私域 = 月入10w+（4.8万）
- Codex 指令入门教程 + 5个神级Skill（6万）

**关键技巧：**
- 视觉上简洁有力，一眼看懂
- 等号右边的结果要足够诱人
- 元素之间要有"意外组合感"（邪修+私域）
- 用加号和等号的视觉格式，区别于普通标题

**适用内容类型：** 方法论总结、组合策略、公式化打法

## 10. 紧急行动型

**核心公式：** `[紧迫感] + [具体行动] + [利益点]`

**变体公式：**
- `[紧急词]！[受众群体]，立刻[行动]`
- `刷到这[内容]，请立即[行动]！`
- `好消息：[限时福利]`
- `就在[时间]，[行动]免费[开放/获取]`

**真实案例：**
- Damn！所有AI开发者，立刻停下npm install（57万）
- 刷到这套提示词，请立即搬运到小红书和闲鱼！（11万）
- 好消息：自动获得ChatGPT Team两个月免费（55万）
- 就在昨天，我把AI热点监控网站免费开放给所有人了（7.5万）

**紧迫词库：**
- 时间紧迫：立刻、马上、赶紧、就在昨天
- 行动指令：停下、搬运、收藏、保存
- 稀缺暗示：限时、免费开放、仅剩

**关键技巧：**
- 动作词要明确直接
- "限时/限量"制造稀缺性
- 配合"免费"降低行动成本
- 适合福利分享、安全预警、限时活动

**适用内容类型：** 限时福利、安全预警、资源分享

## 11. 对比VS型

**核心公式：** `[A] vs [B] + [结论/清单]`

**变体公式：**
- `[产品A] vs [产品B]：[对比维度]`
- `[国家/地区A][类型] vs [国家/地区B][类型]：[清单]`
- `[角度A]vs[角度B]：[结论]`

**真实案例：**
- 中国AI应用 vs 美国工具：对标清单
- 硬件vs软件：AI超级周期下半场真正机会（22万）
- gemini现在是美国版豆包 中看不中用（45万）

**关键技巧：**
- 对比双方都要有知名度
- 结论要鲜明不和稀泥
- 配合清单/表格增加实用价值
- 中外对比、新旧对比天然有话题性

**适用内容类型：** 产品对比、工具评测、行业分析

## 12. 禁忌边界型

**核心公式：** `[禁忌暗示] + [工具] + [突破方法]`

**变体公式：**
- `[N]个[工具]提示词感觉非法（但不违法）`
- `探索[工具]边界：[极端用法]`
- `[工具]有一个隐藏模式...[夸张能力]`
- `用[方法]突破[工具]限制`

**真实案例：**
- 这10个ChatGPT提示词感觉非法（但不违法）（42万）
- 探索GPT image 2边界：13个字生成情趣内衣电商图（66万）
- ChatGPT有一个隐藏模式。它可以像超人一样思考（11万）
- 用Hermes/Flowith突破GPT image 2限制

**禁忌暗示词库：**
- 非法（但不违法）、隐藏模式、突破限制
- 边界探索、破解、解锁
- "别人不敢教的""官方不会告诉你的"

**关键技巧：**
- 括号补充"但不违法"消除顾虑、增加反差
- "隐藏""突破"激发猎奇心理
- 流量最高但也最容易翻车，注意尺度
- 不要做虚假的禁忌暗示，内容要能兑现

**适用内容类型：** 提示词分享、功能探索、尺度测试

## 通用爆款元素速查

| 元素 | 说明 | 优先级 |
|------|------|--------|
| 具体数字 | 金额、数量、倍数、天数 | ★★★★★ |
| 省略号悬念 | "..."制造未完待续感 | ★★★★ |
| 感叹号强化 | 情绪型标题标配 | ★★★★ |
| 口语化表达 | 卧槽>震惊，完犊子>严重缺陷 | ★★★★★ |
| 工具名/产品名 | 增强可搜索性 | ★★★★ |
| 低门槛+高收益 | 永恒的流量密码 | ★★★★★ |
| 第一/二人称 | "我""你""我朋友" | ★★★★ |
| 括号补充 | 末尾补充关键信息 | ★★★ |
| 免费/开源 | 降低行动门槛 | ★★★★ |
| 时间锚点 | 就在昨天、2026年 | ★★★ |
